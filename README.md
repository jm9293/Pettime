# Pettime
애견 미용 출장 서비스 Pettime
>Pettime은 애견 미용 출장서비스를 쉽게 예약할수 있는 웹서비스입니다.

## 기능목록
1. 일반회원 페이지

   회원가입 후 에견미용을 예약할수 있고 공지사항 열람, QNA 게시글 작성등이 가능합니다.

2. 관리자 전용 페이지

   관리자 페이지 회원관리, 디자이너관리 , 게시글 관리등을 할 수 있습니다.

3. 디자이너  전용 페이지

   디자이너의 일정등을 열람하고 관리 할 수 있습니다.

## 개발정보
- 언어 : JAVA 1.8 , JavaScript(ES6) , HTML5 , CSS3 , Oracle SQL , JSP 
- WAS : Tomcat 9.0.38
- DBMS : Oracle XE 112
- Front-end FrameWork : Vue.js (일반회원 페이지) 
- Back-end FrameWork : Spring 
- Project-Manager : Maven, npm
- API : 카카오지도 , 오픈웨더 날씨 , 도로명주소검색 ,  아임포트 결제모듈
- Lib : COS , JSTL , MAIL , jQurey3.3.1 , json simple, Bootstrap 4.3.1, BootstrapVue , moment.js , Vuex ,  Router, Loadscript , Mybatis
